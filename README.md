# Portfolio

A high-performance, responsive personal portfolio website designed to showcase software engineering skills, projects, and professional experience. Built with modern web technologies focusing on performance, accessibility, and clean architecture.

## Technologies Used

- **Frontend Framework**: React 18, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, PostCSS
- **Animations**: Framer Motion, Anime.js
- **3D Graphics**: Three.js, React Three Fiber
- **Form Handling**: EmailJS
- **Icons**: Lucide React

## Key Features

- **Responsive Design**: Fully adaptive layout for all screen sizes.
- **Performance Optimized**: Fast load times with code splitting and lazy loading.
- **Interactive UI**: Smooth animations and 3D elements for enhanced user experience.
- **Contact Integration**: Functional contact form connected via EmailJS.
- **Type Safety**: Comprehensive TypeScript implementation for robustness.

## Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Piyush64-bit/MyPortfolio.git
    cd MyPortfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Environment Configuration**
    Create a `.env` file in the root directory and add your EmailJS credentials:
    ```env
    VITE_EMAILJS_SERVICE_ID=your_service_id
    VITE_EMAILJS_TEMPLATE_ID=your_template_id
    VITE_EMAILJS_PUBLIC_KEY=your_public_key
    ```

4.  **Start the development server**
    ```bash
    npm run dev
    ```

5.  **Build for production**
    ```bash
    npm run build
    ```

## Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Main application views.
- `src/assets`: Static assets and media.
- `src/context`: React context definitions.
